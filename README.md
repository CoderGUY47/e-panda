# 🐼 expert | modern retail experience

this is a high-end web shop made with simple tools but expert design. it uses a dark and premium look to show products clearly and fast. 🚀

## 📦 what is inside
- **fast loading** ⚡: we use loading cards so the user never sees a blank screen.
- **smart data** 💎: it gets products from a real api (fakestoreapi) but makes them look like luxury items.
- **user protocols** 👤: a full system to manage addresses, profile info, and security.
- **shopping flow** 🛒: items go to a cart or wishlist with smooth animations.
- **mobile first** 📱: every part of the site works perfectly on phones and big screens.

## 🛠️ tech used
- **html5**: for the structure.
- **tailwind css**: for the fast and clean design.
- **daisyui**: for the premium components.
- **vanilla javascript**: for all the logic and store management.
- **localstorage**: to keep your data even if you refresh the page.

## 🏁 how to run
1.download the files.
2.open index.html in any browser.
3.wait for 2.5 seconds for the loader to finish.

## 🎨 design notes
the project uses a mix of glass-morphism and technical grids. comments are kept in lowercase for a clean human look.
